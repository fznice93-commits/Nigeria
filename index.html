Background

项目背景与分析方法

PDF 的核心问题是：AI 生图在本地化审美上缺乏标准，容易生成“不对味”的尼日利亚人物，因此需要用真实样本建立可复核的人物形象框架。

为什么从 Lagos 切入

Lagos 集中体现尼日利亚的时尚、美妆、音乐、影视和社媒审美。它既有都市商业人物，也有 KOL、娱乐、街头生活和传统视觉符号，因此可作为第一阶段观察入口。

样本与维度

样本按明星/KOL、普罗大众、广告模特三类组织，并进行男女分组观察。两大模块分别记录结构性特征（五官、发型、皮肤、身材）与文化性特征（妆容、服饰、配饰）。

Sample Map

样本来源图谱

样本来源图谱展示 KOL / 明星、普罗大众、广告模特三类人群的实际视觉差异：KOL 更精致，普罗大众更生活化，广告模特更标准化。

KOL / 明星男性

公众人物样本更偏社媒化与造型完整，发型、胡须、服装和拍摄姿态都更受控。男性样本中可观察到短发、胡须、墨镜、皮衣或西装等更强的形象管理。

普罗大众男性

普罗大众男性样本更生活化，服装、姿态和摄影环境保留了街头、市场或新闻现场的自然状态。相较广告模特，表情和构图更少精修控制。

广告模特男性

广告男性样本更标准化，常与手机、现金、App 机制或金融服务信息同框。表情和动作更明确，骨相与发型也更容易被商业化整理。

KOL / 明星女性

女性 KOL 样本的妆发和服装更精致，假发、长直发、编发、礼服和精修妆面出现频率更高。它们能提供审美上限，但不能替代普通人样本。

普罗大众女性

普罗大众女性样本更能观察日常穿衣、头巾、街头行走姿态和真实环境。它们对避免资产库过度 KOL 化非常重要。

广告模特女性

广告女性样本的构图、背景和动作更标准化，常见手机、二维码、明亮色块和亲和表情。它们能帮助观察商业化人物如何处理发型、肤色和姿态。

总判断

五套分析最一致的结论是：尼日利亚本地化人物不是“深肤色 + 宽鼻 + 厚唇”的简单标签，而是五官骨相、发型系统、皮肤色阶与质感、体型范围、妆容影响、服饰场景和配饰边界共同构成的视觉体系。

Module 1

结构性特征

结构性特征指人物本身的生理和物理特征，按脸部、头发、皮肤、身材的观察顺序组织。

结构性特征 1/4

五官

尼日利亚人物五官呈现清晰骨相与多样脸型并存的特征：椭圆脸、长椭圆脸和方圆脸都可见；鼻翼较饱满、鼻根不过度拔高、唇形饱满是更稳定的跨性别观察点。

1. 骨相先行

清晰骨相是尼日利亚模特组和广告样本中常见的上位结构特征，表现为额头、颧骨、鼻梁、下颌、颈部与肩线之间的轮廓关系清楚，但它不等同于某一种具体脸型。

五官男女并列证据

模特组清晰骨相

观点说明：广告模特组常更强调骨相、颧骨和脸部明暗关系。男女案例并列后，可以把“清晰骨相”作为上位结构，而不是某一种性别或脸型的专属特征。

图片说明：左图男性样本显示额头、颧骨、鼻梁和下颌的硬朗转折；右图女性样本显示颧骨、下颌和颈肩线条的清晰关系，二者共同支撑模特组骨相清晰这一共性判断。

结论判断：清晰骨相是五官维度的上位描述，后续脸型、下颌、鼻唇特征都应在这个结构基础上判断。

2. 脸型多样性

在骨相基础上再看脸型。椭圆脸、长椭圆脸、方圆脸都可能出现，这是三种可以区分的脸型倾向：椭圆脸更接近鸡蛋形比例，长椭圆脸的纵向比例更明显，方圆脸则强调面部下半区宽度和下颌重量；下颌线、下颌宽度和下颌重量只作为脸型下半部分的补充刻画。

椭圆脸

观点说明：椭圆脸的判断重点是额头、颧骨、下颌线条较柔和，整体比例接近鸡蛋形，脸部宽窄过渡顺而不强调纵向拉长。

图片说明：左图男性样本面部轮廓较平顺，额头、颧骨和下颌没有明显棱角；右图女性样本脸部宽窄过渡柔和，可用于支撑普通椭圆脸判断。

结论判断：椭圆脸是尼日利亚人物脸型范围之一，应与长椭圆脸、方圆脸分开标注。

长椭圆脸

观点说明：长椭圆脸和普通椭圆脸的区别在于脸部纵向比例更长，从额头到下巴的距离更明显，脸宽相对收窄，整体不是单纯柔和轮廓，而是偏纵向拉伸。

图片说明：左图男性样本为正面短发近景，额头到下巴的纵向距离较明显、脸宽相对收窄；右图女性样本同样突出额头、面中和下巴的纵向关系，两者共同支撑长椭圆脸判断。

结论判断：长椭圆脸应作为独立脸型标签保留，和普通椭圆脸分开标注，避免把“柔和轮廓”和“纵向比例偏长”混为一类。

方圆脸

观点说明：方圆脸体现的是面部下半区宽度、下颌重量和整体轮廓的组合。男女案例并列后，可以说明“方圆”是脸型多样性的一部分，而不是只属于男性角色。

结论判断：方圆脸是脸型多样性的一部分，不能把下颌重量直接等同于独立分类。

3. 鼻型与唇形共性

鼻翼和唇形是跨性别的五官共性特征，应单独描述，但仍需避免把它们写成单一刻板脸。

结构性特征 2/4

发型

尼日利亚男性发型以短寸、渐变、光头、短辫和清晰发际线为主，胡须打理也需要一起观察；女性样本中假发、长直发、波浪卷、编发和盘发更常见。头巾、Gele、帽子是服饰配饰，不计入发型本身。

男性发型体系

男性样本的发型重点集中在可打理毛发结构：短寸、渐变、光头、短辫、清晰发际线和胡须造型共同构成整洁度与身份感。

皮肤

Final Findings

结论性报告

以下总结用于后续 prompt 标签词库、LoRA 素材库和本地化人物标准沉淀，不进入具体广告投放判断。

1. 稳定特征

  • 共性结论：肤色跨度明显，不可单一化；色阶、底调、光泽和肤质层次需要共同标注。
  • 共性结论：五官应先看清晰骨相，再看多样脸型、鼻翼/鼻根与饱满唇形；下颌只是脸型下半部分的补充刻画。
  • 共性结论：深肤色表现要保留自然光泽、暗部层次和真实肤质，而不是人为叠加强高光或磨成平面。
  • 差异结论：发型体系需要男女分区，男性短发/渐变/胡须打理与女性假发/编发/盘发不能混排。
  • 差异结论：服饰装扮和体型会随身份、场景、样本类型变化；传统符号、Gele、珠饰、彩绘和疤痕不能作为普通人物默认配置。

2. 三类样本差异

  • KOL / 明星：更精致、更社媒化，更重妆发和造型。
  • 普罗大众：更自然、更生活化，穿衣和肤质更真实。
  • 广告模特：骨相更清晰，造型更时装化，画面更标准化。

3. 后续资产库建议

  • 将两大模块、七个子项继续拆成 prompt 标签词库。
  • 筛选高代表性图片作为 LoRA 训练素材。
  • 建立正向标签和负向标签，避免刻板化生成。
  • 按 KOL、普罗大众、广告模特分别建立三套人物模板。
  • 继续补充不同年龄、体型和职业状态的普通人样本。
